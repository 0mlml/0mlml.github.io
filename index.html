Go

Some of my projects in Go:

    Downloads all messages from a discord server. Uses the discord API. Curiously I spent more time on the logging than the actual functionality.
  • Chesshook intermediary
    Allows chesshook to communicate with an external engine running on the same or different machine through the websocket protocol. I tried to make it not destroy itself if many people try to use it at once, but I have not tested it as I don't see it necessary for my use.
